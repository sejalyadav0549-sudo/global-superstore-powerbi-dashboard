# 🌍 Global Store Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive Power BI dashboard built to analyze global retail store data. It provides insights into sales, profit, customer segments, and regional performance to support data-driven decision-making.

## 🎯 Objectives
- Analyze overall sales and profit performance
- Compare region-wise and country-wise sales
- Identify top-performing products and categories
- Understand customer segment contribution
- Track sales trends over time

## 🛠️ Tools & Technologies
- Power BI  
- Power Query (Data Cleaning & Transformation)  
- DAX (Basic Measures & Calculations)  
- Data Visualization  

## 📊 Dashboard Features
- KPI Cards (Total Sales, Profit, Quantity)
- Region & Country-wise analysis
- Category & Sub-category breakdown
- Customer segment analysis
- Time-based sales trends (monthly/yearly)
- Interactive filters and slicers

## 📈 Key Insights
- Identified top-performing regions contributing highest revenue
- Observed profit variations across categories
- Found key customer segments driving sales
- Analyzed seasonal trends in global sales data

## 🚀 Conclusion
This dashboard helps in understanding global retail performance and supports strategic business decisions. It demonstrates strong data analysis and visualization skills using Power BI.
